# LuhutOS

LuhutOS is an Arch-based distribution with GNOME Wayland, safe GPU handling, and macOS-like polish.

## Directory Structure

- `iso/` - Archiso profile (copy of releng)
- `packages/` - PKGBUILD sources for luhutos-* packages
- `installer/` - luhutos-installer TUI source
- `repo/` - Built package repo artifacts
- `docs/` - Documentation
